# 保险合同解释助手

一个基于 Web 的智能保险合同解释助手，帮助用户理解保险合同条款，提供专业的保险咨询服务。

## 项目结构

```
InsuranceHtml/
├── Index.html              # 主页面文件
├── README.md               # 项目说明文档
├── css/                    # 样式文件目录
│   └── style.css          # 主样式文件
├── js/                     # JavaScript文件目录
│   ├── config.js          # 配置文件
│   └── main.js            # 主逻辑文件
├── assets/                 # 静态资源目录
│   ├── images/            # 图片资源（预留）
│   ├── fonts/             # 字体文件（预留）
│   └── icons/             # 图标文件（预留）
└── icon/                  # 原有图标目录
```

## 文件说明

### HTML 文件

- `Index.html`: 主页面，包含完整的 UI 结构

### CSS 文件

- `css/style.css`: 主样式文件，包含所有 UI 组件的样式定义
  - 使用 CSS 变量定义主题色彩
  - 响应式设计支持
  - 动画效果和过渡
  - 模块化样式组织

### JavaScript 文件

- `js/config.js`: 配置文件

  - API 配置
  - 文件上传配置
  - UI 配置参数
  - 错误消息定义

- `js/main.js`: 主逻辑文件
  - InsuranceAssistant 主类
  - 事件处理逻辑
  - 文件上传功能
  - 拖拽支持
  - 工具函数

## 使用方法

### 一键启动（推荐，含后端代理）

在 `InsuranceHtml_ui_opt` 目录运行：

```bash
PYTHON_BIN=/data2/wangliangmin/miniconda3/envs/baichuan-chat/bin/python ./run_ui_opt.sh
```

默认访问地址：`http://127.0.0.1:8090`

说明：
- 该脚本会先启动 `insurance_api/main_api.py` 对应的 FastAPI 服务；
- 再启动前端静态服务与 `/api/v1/*` 代理，前端可直接连后端；
- 页面顶部会显示后端连接状态。

### 开发环境

1. 克隆或下载项目文件
2. 使用 Web 服务器打开`Index.html`
3. 或直接在浏览器中打开文件

### 生产环境

1. 将所有文件上传到 Web 服务器
2. 配置`js/config.js`中的 API 地址
3. 访问主页面开始使用

## 自定义配置

### 主题色彩

在`css/style.css`中修改 CSS 变量：

```css
:root {
  --green: #4c742c; /* 主色调 */
  --green-dark: #395b1f; /* 深色调 */
  --gray-light: #f5f5f5; /* 浅灰色 */
  --gray-medium: #d9d9d9; /* 中灰色 */
}
```

### 文件上传限制

在`js/config.js`中修改配置：

```javascript
upload: {
    maxSize: 10 * 1024 * 1024,  // 最大文件大小
    allowedTypes: [...]         // 允许的文件类型
}
```
